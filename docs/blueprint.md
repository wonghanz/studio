# **App Name**: NATIVE Learn

## Core Features:

- User Authentication & Profile: Secure user registration and login using email/password and Google Sign-in. Allows users to select their target exam (MUET/IELTS) for personalized learning paths and access to saved progress.
- Daily Study Journey: Engage with story-based 'Daily Diary' content. Fetches and displays reading stories, integrated with comprehension questions (MCQ, T/F/NS) to auto-mark answers and provide explanations. This also sets the foundation for future listening exercises.
- AI Writing Feedback: A text editor for writing practice. Users can save drafts locally in the browser (for offline-like functionality) and submit essays to the backend for AI evaluation. The AI tool will provide rubric-based feedback and a band score, with the capability to sync when online.
- AI Speaking Examination: An integrated browser-based audio recorder. Users can record their speech, which is then uploaded for AI analysis. The AI tool provides examiner-style feedback including a band score, fluency tips, coherence feedback, and model Band 5 answers.
- Personalized Progress Dashboard: Visualize academic growth through a dashboard displaying band score trends for both writing and speaking. Provides a weakness profile, identifying areas like grammar, coherence, vocabulary, and fluency across different skills.
- AI Daily Task Planner: Generates AI-suggested daily study tasks (e.g., specific writing prompts, speaking topics, reading passages, or listening exercises) tailored to the user's progress and selected exam target (MUET/IELTS). The AI tool will adapt recommendations based on performance data.

## Style Guidelines:

- Primary color: `#26B2D8` (A vibrant, clear blue-green, representing knowledge and digital clarity, chosen for active elements and branding.)
- Background color: `#E0F3F8` (A soft, heavily desaturated blue-green, providing a calm and focus-friendly canvas for a learning application in a light scheme.)
- Accent color: `#0AA380` (A rich, earthy green, providing contrast and signifying progress or confirmation for calls-to-action and key highlights.)
- All text: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring optimal readability for educational content and UI elements.
- Use minimalist, line-based icons for clarity and to maintain a professional yet approachable aesthetic, consistent with a modern educational platform.
- A clean, card-based layout structure for the dashboard, promoting clear information hierarchy and intuitive navigation. Content screens will prioritize readability with ample white space.
